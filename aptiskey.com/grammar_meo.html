Gõ để lọc phần nội dung phù hợp.
Mẹo nhanh: Khi gặp bài chọn từ loại, ưu tiên soi vị trí đứng (trước/ sau danh từ, sau động từ to be, trước danh từ số nhiều/không đếm được…) để quyết định noun/adj/adv.

Về cơ bản thì ngữ pháp này không tính điểm nên bạn có thể học sơ qua thôi nhé! nhưng nó cũng là một cơ sở để giám khảo chấm điểm viết và điểm nói của bạn.

Tổng hợp toàn bộ mẹo grammar
Noun: a/an/the + N, Adj + N, Prep + N/V-ing
Đếm được số nhiều: few, a few, many, several, a number of…
Không đếm được: much, little, a little, a great deal of…
Cả hai: some, a lot of, lots of, all.
Adj: a/an/the + adj + N, No + adj + N, Become + adj
V-ed = bị động (interested), V-ing = chủ động (interesting).
Adv: thường adj + ly. Vị trí: be + adv + V-ing/V-ed, aux + adv + V, đầu câu Adv, S + V.
S–V: Each/Every/One of → số ít; A number of → số nhiều; The number of → số ít; Either/Neither → theo B.
Rel. clause: who (người), whose (sở hữu người), which (vật/ mệnh đề), S, V, which V (số ít).
If: 0 hiện tại–hiện tại; 1 hiện tại–will/can V; 2 were/ V2 – would/could V; 3 had V3 – would/could have V3.
So sánh: bằng as…as; không bằng not as…as; hơn adj ngắn + er than / more + adj dài + than; nhất the + adj-est / the most + adj. Bất quy tắc: good→better→best; bad→worse→worst; many/much→more→most; little→less→least.

I. Mẹo về Danh từ (Noun)

Cấu trúc/ Kết hợp thường gặp
  • Giới từ + N/ V-ing
  • The + N
  • A/ an + N (đếm được số ít)
  • Each + N (đếm được số ít)
  • Tính từ sở hữu/ sở hữu cách + N
  • Ngoại động từ + N (tân ngữ)
N đếm được số nhiều
  • There are, few, a few, a number of, the number of
  • many, several, several of, a variety of
N không đếm được
  • much, little, a little
  • a great deal of, a large amount of
Dùng cho cả hai: some, some of, a lot of, lots of, all.
Lưu ý: N đếm được số ít bắt buộc có a/ an/ the đứng trước.

II. Mẹo về Tính từ (Adjective)

Vị trí & kết hợp
  • A/ an/ the + adj + N
  • Từ chỉ số lượng + adj + N
  • Tính từ sở hữu/ sở hữu cách + adj + N
  • Tính từ sở hữu/ sở hữu cách + most + adj(dài) + N
  • The most + adj(dài) + N
  • No + adj + N
  • This/ that/ these/ those + adj + N
  • Become + adj/ N
Ba dạng tính từ
  • Tính từ thường: useful, beautiful, special…
  • V-ed/ V3: nghĩa bị động (chịu tác động) – interested, inspired
  • V-ing: nghĩa chủ động/ bản chất – interesting, amazing

III. Mẹo về Trạng từ (Adverb)

Cấu tạo

Thường: adj + ly (usefully, extremely…).

Mẹo đáp án: Nếu thấy cặp “A” và “A+ly” → “A” là adj, “A+ly” là adv.

Vị trí thường gặp
  • Be + adv + V-ing / V-ed
  • Trợ động/ khuyết thiếu + adv + V
  • Trợ động + adv + be + V-ed
  • Have/ has/ had + adv + V-ed
  • Be + adv + adj
  • Đầu câu: Adv, S + V
  • Cụm lượng từ + adv + adj + N

IV. Mẹo về Sự hòa hợp giữa Chủ ngữ và Động từ

Chia số ít (is/ was/ has/ V-s,es) khi chủ ngữ là:
  • N đếm được số ít/ không đếm được
  • One of the + N(số nhiều)
  • Danh động từ (V-ing)
  • Đại từ bất định (someone, everyone…)
  • Each/ Every + N(số ít)
  • The number of + N(số nhiều)
  • Much, little, a little, a great deal of… + N(không đếm được)
  • He/ She/ It
Chia số nhiều (are/ were/ have/ V) khi chủ ngữ là:
  • N đếm được số nhiều
  • The + adj (chỉ nhóm người: the poor, the rich…)
  • A number of + N(số nhiều)
  • A and B; Both A and B
  • few, a few, many, several, several of, a variety of + N(số nhiều)
  • You/ We/ They
Đặc biệt
  • Some/ some of/ a lot of/ lots of/ all + N(không đếm được) → số ít
  • Some/ some of/ a lot of/ lots of/ all + N(đếm được số nhiều) → số nhiều
  • A + giới từ + B → chia theo A
  • Either A or B; Neither A nor B → chia theo B

V. Mẹo về Mệnh đề quan hệ

  • Người: N + who + V
  • Người (sở hữu): N + whose + N
  • Vật/ mệnh đề: N + which + V/ mệnh đề
  • Toàn mệnh đề trước đó: S + V, which + V (số ít)

VI. Mẹo về Câu điều kiện

Loại 0 
If + S + V (hiện tại), S + V (hiện tại)
Example: If you are able to finish this task by this afternoon, you are clever.
Loại 1 
If + S + V (hiện tại), S + will/ can + V
Example: If it does not rain, I will go shopping.
Loại 2 
If + S + were + …, S + would/ could + V
Hoặc: If + S + V (quá khứ đơn), S + would/ could + V
Example: If I were you, I would marry her.
Loại 3 
If + S + had + V3, S + would/ could + have + V3
Example: If you had helped me yesterday, I could have passed the test.

VII. Mẹo về So sánh

So sánh bằng/ không bằng 
S + be/ V + as + adj/adv + as + S
S + be/ V + not as + adj/adv + as + S
Mẹo: “as … as”, “not as … as”.
So sánh hơn 
Adj ngắn: adj + er + than
Adj dài: more + adj + than
Nhấn mạnh: much/ far/ a lot + dạng so sánh hơn.
Đặc biệt
  • V + more + N + than…
  • The + so sánh hơn, the + so sánh hơn (càng… càng…)
  • Of the two + N(số nhiều) → S + be/ V + the + so sánh hơn
So sánh nhất & bất quy tắc 
Adj ngắn: the + adj + est
Adj dài: the most + adj
Bất quy tắc: good/well → better → the best; bad/ill → worse → the worst; many/much → more → the most; little → less → the least.
